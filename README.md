# 📊 Clustering in Machine Learning

## 📌 Overview

This repository contains implementations of clustering algorithms using unsupervised machine learning techniques. The goal is to discover hidden patterns and group similar data points without labeled outputs.

---

## 🧠 What is Clustering?

Clustering is a type of **unsupervised learning** where data is grouped into clusters based on similarity. Unlike classification, clustering does not require predefined labels.
